Finalizing and ending a completed beamtime for a user¶

Required Information¶

Goals of the Process¶

  1. Ensure that the Export directory is empty. If not, delete everything in Export. Confirm before doing this.
  2. Tar and zip the entire xpdUser tree with name <PIlastname>_<saf#>_<date>.tar.gz
  3. Place a copy of the tarball in a remote archive, and in the Export directory
  4. Verify that the archival copy is present and can be accessed
  5. Delete everything except the Export directory
  6. Keep operator informed of what is going on
